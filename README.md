# 動画概要
リポジトリの動画概要を作成します。
現時点ではGitHubのみの対応です。

## 設定
.env.exampleに従ってください。

## 起動方法
```shell
git clone https://github.com/makaseloli/zatugaku.git
uv sync
.venv/bin/python main.py
```

## 使用方法
左のタブから順に実行してください。

## 推奨スペック
- VRAM32GB以上
- RAM64GB以上
  
> [!TIP]
> メモリさえあればCPUでも動きます。

## 使用したもの
### 主要なライブラリ
- Diffusers
- Gradio
- LlamaIndex
- Open JTalk

### 開発に使用
- VSC
- GH Copilot
